Description

A bare HTML page without styles doesn't look very presentable. There is just so much that can be done! For example, you can add a background, margins, and paddings, or change the color, size, or location of the elements. In this stage, you will transform the appearance of your site and make it visually impressive.

Don't forget to connect the CSS file to HTML, otherwise, the properties will not apply.



If you'd like, you can try making the same site design as in the examples below. At this link you can find the images that were used to create this site.

Objectives

  1. Reset default margins and paddings so that all the elements are displayed with the same indentation in different browsers.

  2. Use CSS Grid or Flexbox to rearrange the elements of the web page as you like.

  3. Change the height of the header of your site.

  4. Change the background color of the site header or set a cool background image.

  5. Set the height and width of the images on your site.

  6. Change the height of the footer of your site.

  7. Change the background color of the site footer.

  8. Add styles to other elements: this is up to you! You can also create additional HTML elements if you wish.
